# 🏢 Employee Attrition Analysis

## 📌 Project Overview
This project analyzes IBM HR data to find why employees leave 
the company and predicts future attrition using Machine Learning.

## 🛠️ Tech Stack
- **Python** - Data Analysis & ML
- **MySQL** - Database & SQL Queries
- **Pandas** - Data Manipulation
- **Matplotlib & Seaborn** - Data Visualization
- **Scikit-learn** - Machine Learning

## 📊 Business Questions Solved
1. Which department has highest attrition rate?
2. Does salary affect employee attrition?
3. Do overtime employees leave more often?

## 🔍 Key Findings
-  **Sales Department** has highest attrition (20.63%)
-  Employees who left had **42% lower salary** than those who stayed
-  Overtime employees are **more likely to quit**

## 🤖 ML Model
- **Algorithm:** Logistic Regression
- **Accuracy:** 88%+
- **Feature:** Predicts if an employee will quit or stay
